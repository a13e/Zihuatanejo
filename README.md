## Whats' this?
___会いきてくれた人へギフトを添えてキモチを伝える___

## Why we gonna create this?
* Giftの会社として__新しい、面白いコト__をしたい。
* Giftを贈るということを__もっと広く考えてみるキッカケ__をつくる。
* __Gift文化__というものを会社として表現させるため。
* __モノづくり__文化をもっと強めたい。

## How we gonna do that?
* オフィス入り口に来客の人を受け付ける`iPadアプリ`をつくる。
* `名前`を入力、`会いたい人`を選択してもらい、こちらが感謝のキモチを伝えたい相手ならその___ギフトを贈れる___。
* もしくはMTG後に相手のメールアドレス宛に___Giftメールが送信される。___

## One more thing
* はじめは社内向けとしてつくるが、パッケージとしてtoBサービス展開もしたい。

  * 「来客」という文脈からのローカル店舗訴求
